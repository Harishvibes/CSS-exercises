# CSS Simple Exercises

Welcome to my CSS practice repository! 🎨

This repository contains a collection of simple CSS exercises that I created while learning and improving my frontend development skills. 
The goal is to practice core CSS concepts through small, focused projects and examples.

## Topics Covered

* CSS Selectors
* Colors and Backgrounds
* Typography
* Box Model
* Margins and Padding
* Borders and Border Radius
* Display Properties
* Positioning
* Flexbox
* CSS Grid
* Transitions and Animations
* Pseudo-Classes
* Pseudo-Elements

## Purpose

This repository serves as:

* A personal learning journal
* A collection of CSS practice exercises
* A reference for beginners learning CSS
* A place to experiment with different styling techniques

## Folder Structure

```text
css-simple-exercises/
│
├── exercise-01/
├── exercise-02/
├── exercise-03/
├── exercise-04/
└── ...
```

Each exercise contains its own HTML and CSS files demonstrating a specific concept.
